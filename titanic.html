Aim

The aim of the this project is to analyze the given train data in file 'train.csv' and then make a prediction model testing it on file 'test.csv

Procedure Followed:

  • Import the data from files into pandas dataframe
  • Visualize the data using various graphs extraing out important features
  • Feature Engineering - making data ready for prediction model by replacing strings with numerical values and removing unneccesary columns

def age_category(age):
    if age <= 16:
        age = 0
    
    elif age > 16 and age <= 26:
        age = 1
        
    elif age > 26 and age <= 36:
        age = 2
        
    elif age > 36 and age <= 62:
        age = 3
        
    elif age > 62:
        age = 4
        
    return age
